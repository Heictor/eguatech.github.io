A brief look at what makes Egua special.

Anatomy of Egua

Here is some of Egua syntax.

• Strings are given with double " ".
• Concatenation is done with +.
• No typed variable required to assign string to it.

An example, using simple string structure:

• Egua accepts basic +, -, / and * operations.
• Exponent is given with **. 2 ** 2 = 4.
• Module is given by %.
• The arithmetic of Egua is linear from left to right, so it does not use the mathematical priority rules. WATCH OUT!
• Parentheses do not require mathematical operation.

The online IDE has a sample session, which contains code / scripts previously developed for quick testing and simplified learning of the Egua language.
• Hello, World, Strings, Math, Conditional, Looping, Functions, Arguments, Fibonnaci, Lists and Factorial.

Egua Console

With the console you can run your code and check the script output. It's very simple, just hit Run.
